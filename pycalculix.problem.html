pycalculix.problem module¶

This module stores the Problem class which is used to solve different types of analysis.

class pycalculix.problem.Problem(feamodel, problem_type, parts, fname='')[source]¶

Bases: pycalculix.base_classes.Idobj

Makes a problem which can be analyzed with Calculix ccx.

Parameters:
  • feamodel (FeaModel) – the parent FeaModel
  • problem_type (str) –

    model type, options:

    • ‘struct’: structural
  • parts (Part or list of Part) – stores the parts to run the analysis on
  • fname (str) – file prefix for the problem .inp and results files If value is ‘’ it will default to the project name of the FeaModel
