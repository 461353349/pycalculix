class pycalculix.geometry.Arc(p1, p2, actr, sign=1)[source]¶

Bases: pycalculix.base_classes.Idobj

Makes an arc from points p1 to p2 about center actr.

Arcs should be 90 degrees maximum.

Parameters:
  • p1 (Point) – first point
  • p2 (Point) – second point
  • actr (Point) – arc center
  • sign (int) – 1 or -1, defines direction of line +1: p1 -> p2 -1: p2 -> p1

get_points()[source]¶

Returns all the child points in the area.

These are the points that define all lines and arcs in the area. Does not return arc center points, only line start and end points. Only unique points are returned (no duplicate points). Points are generated by iterating through the line_list.
